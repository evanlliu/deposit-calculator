# 定存计算器 / Deposit Calculator

一个轻量级的定期存款计算工具，支持正向计算和反向推算，可用于快速计算定存年利率、利息、本金和期限。

当前版本：**v1.5.0**

---

## 功能特点

- 支持四种计算模式：
  - 求年利率
  - 算利息
  - 求本金
  - 求期限
- 支持开始日期、结束日期自动计算存款天数
- 支持年化天数口径切换：
  - 365 天
  - 360 天
- 支持中文 / English 双语言切换
- 支持 PC 和移动端自适应布局
- 支持 iOS Safari 添加到主屏幕
- 支持 PWA 离线缓存
- 无需后端，直接静态部署即可使用

## 使用说明

### 1. 求年利率

已知：

- 本金
- 存款期限
- 利息

点击 **开始计算** 后，系统会自动反推出年利率。

公式：

```text
年利率 = 利息 ÷ 本金 ÷ 存款天数 × 年化天数 × 100%
```

---

### 2. 算利息

已知：

- 本金
- 存款期限
- 年利率

点击 **开始计算** 后，系统会自动计算利息。

公式：

```text
利息 = 本金 × 年利率 ÷ 100 × 存款天数 ÷ 年化天数
```

---

### 3. 求本金

已知：

- 存款期限
- 年利率
- 利息

点击 **开始计算** 后，系统会自动反推出本金。

公式：

```text
本金 = 利息 ÷ 年利率 × 100 ÷ 存款天数 × 年化天数
```

---

### 4. 求期限

已知：

- 本金
- 年利率
- 利息

点击 **开始计算** 后，系统会自动反推出存款期限。

公式：

```text
存款天数 = 利息 ÷ 本金 ÷ 年利率 × 100 × 年化天数
```

---

## 日期计算说明

如果填写了 **开始日期** 和 **结束日期**，系统会自动计算两个日期之间的天数，并填入存款期限。

当前日期计算逻辑：

```text
存款天数 = 结束日期 - 开始日期
```

例如：

```text
2026-05-01 到 2026-05-02 = 1 天
```

## 年化天数说明

不同银行或产品可能使用不同的年化天数口径：

| 年化口径 | 说明 |
|---|---|
| 365 天 | 常见银行计算方式 |
| 360 天 | 部分产品或银行可能使用 |

可以根据银行产品说明切换。

## PWA 使用说明

本项目支持 PWA，可以在手机上添加到主屏幕。

### iPhone / iOS Safari

1. 使用 Safari 打开网页
2. 点击底部分享按钮

## 更新部署说明

每次修改版本时，建议同步更新以下文件：

```text
index.html
sw.js
manifest.webmanifest
```

其中：

- `index.html` 中显示页面版本号
- `sw.js` 中维护缓存版本号
- `manifest.webmanifest` 中维护应用名称和 PWA 配置

如果版本号没有同步更新，手机或浏览器可能会继续加载旧缓存。
